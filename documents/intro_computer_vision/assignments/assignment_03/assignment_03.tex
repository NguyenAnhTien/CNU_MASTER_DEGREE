\subsection{Steerable Gaussian Filter}
\matlabscript{mfiles/steerable_gaussian_filter}{Steer Gauss Filter}
\matlabscript{mfiles/main}{Demo program}

We use the term "steerable filter" to describe a class of filters in which a filter of arbitrary orientation is synthesized as a linear combination of a set of "basis filters."

1-D Gaussian filter:
\begin{equation}
    g(x)\ =\ e^\frac{-x^2}{2\sigma^2}
\end{equation}

2-D Gaussian filter:
\begin{equation}
G(x)\ =\ e^\frac{-(x^2\ +\ y^2)}{2\sigma^2}
\end{equation}

The $nth$ derivative of a Gaussian in the $x$ direction as $G_n$. Let $(...)^\theta$ represent the rotation operator, such that, for any function $f(x, y)$, $f^{\theta}(x, y)$ is $f(x, y)$ rotated through an angle $\theta$ about the origin.

The first $x$ derivative of a Gaussian, $G_1^{0 \degree}$, is:
\begin{equation}
    G_1^{0 \degree} = \frac{\partial }{\partial x}e^\frac{-(x^2\ +\ y^2)}{2\sigma^2}
\end{equation}

The same function, rotated 90 degrees, is:
\begin{equation}
G_1^{0 \degree} = \frac{\partial }{\partial y}e^\frac{-(x^2\ +\ y^2)}{2\sigma^2}
\end{equation}

A $G_1$ filter at an arbitrary orientation $\theta$ can be formed by a linear combination of $G_1^{0 \degree}$ and $G_1^{90 \degree}$:
\begin{equation}
    G_1^{\theta} = \cos(\theta)G_1^{0 \degree} + \sin(\theta)G_1^{90 \degree}
\end{equation}
