\matlabscript{example_figure_2_3}{Sample code to demonstrate the usage of imadjust function}

\textbf{\textit{imadjust}} function:
\begin{itemize}
    \item Adjusting image intensity values or colormap.
    \item Syntax: 
    \begin{itemize}
        \item \textit{J = imadjust(I)}
        \item \textit{J = imadjust(I,[low\_in; high\_in],[low\_out; high\_out])}
    \end{itemize}
\end{itemize}

\textbf{\textit{imadjust(f, [0 1], [1 0])}} function do a operations as same as \
\textit{imcomplement} function. The function converts the intensity values \
between 0 and 1 to values between 1 and 0. Therefore, the dark pixel will \
become more darker and the bright pixel will become brighter.\\ 

In the \textbf{\textit{imadjust(f, [ ], [ ], 2)}}, the gamma is greater \ 
than 1, it means the mapping is weighted toward lower (darker) output values.


\textbf{\textit{stretchlim}} function:
\begin{itemize}
    \item Find limits to contrast stretch image.
    \item Syntax: 
    \begin{itemize}
        \item \textit{Low\_High = stretchlim(I)}
    \end{itemize}
\end{itemize}

\end{homeworkProblem}

\matlabscript{example_figure_3_6_b}{Using intrans function}

In our contex, the class of the image is not double. Therefore, the \ 
\textbf{\textit{intrans}} function converts the image to double class.
The first parameter \textbf{varargin{1}} is '\textbf{stretch}', so the \
function apply \textbf{contrast-stretching} transformation function to \
the image. The \textbf{contrast-stretching} transformation function \ 
compresses the input levels lower than \textbf{m} into a narrow range \
of dark levels in the output image; similarly, it compresses the values \
above m into a narrow band of light levels in the output. The output is \
a binary image which has higher contrast. \textbf{\textit{length(varargin)}} \
equals to 3, so m is mean of the image\ and E equals to 0.9.
